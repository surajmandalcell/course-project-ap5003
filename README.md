# Installation Process
### Backend
```
virtualenv venv
source venv/bin/activate # for unix  
venv\Scripts\activate # for windows
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
### Front-end

```
cd clients
pnpm i
pnpm start
```
